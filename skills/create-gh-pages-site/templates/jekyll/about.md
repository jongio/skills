<section class="card">
  <ul>
    <li><strong>Includes</strong> — <code>_includes/header.html</code> is the shared top bar.</li>
    <li><strong>Data files</strong> — <code>_data/nav.yml</code> drives the navigation.</li>
    <li><strong>Collections</strong> — the <a href="{{ '/docs/' | relative_url }}">Docs</a> section is a custom collection.</li>
    <li><strong>Front-matter defaults</strong> — layouts are assigned in <code>_config.yml</code>.</li>
    <li><strong>Liquid filters</strong> — <code>date</code>, <code>relative_url</code>, <code>default</code>.</li>
    <li><strong>Theming</strong> — a persisted light/dark toggle and a GitHub source link in the top bar.</li>
  </ul>
