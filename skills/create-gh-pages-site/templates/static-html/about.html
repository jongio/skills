About this template

The static-html template is the simplest way to put a site on GitHub Pages: hand-written HTML with relative links, deployed by the official Pages workflow.

Why relative links

A project site lives at https://USER.github.io/REPO/. If you link assets with absolute paths like /styles.css, the browser asks for https://USER.github.io/styles.css and gets a 404. Relative links (./assets/styles.css) resolve against the current page, so they work at any base path with no configuration.

Add a page

Drop another .html file next to this one and link to it relatively. That's the whole workflow.
