How to use a template

  1. With the Copilot skill: /create-gh-pages-site and describe the site you want (e.g. "an Astro blog").
  2. Or run the generator yourself: node scripts/new-site.mjs <template> --repo <owner/name>
  3. Push to main, then set Settings → Pages → Source → GitHub Actions.
